# KC INTERNATIONAL 통합경영관리 시스템

식품 유통업 영업·채권·재고·원가 통합 관리 시스템. 단일 HTML 파일로 동작합니다.

> **본 저장소의 모든 데이터는 데모용 가상 데이터입니다.** 거래처명·금액·단가·원가는 실제 값과 무관합니다.

## 실행

`index.html`을 브라우저로 열면 됩니다. 별도 설치·서버 불필요.

**[▶ 데모 보기](https://.github.io//)**

## 구성

10개 대분류 · 54개 화면

| 분류 | 주요 화면 |
|---|---|
| 대시보드 | 6대 KPI, 월별 매출·이익, 미수 연령, 드릴다운 |
| 영업관리 | 견적→주문→출고→매출 파이프라인, 반품, 할인·리베이트, 세금계산서 |
| 수금·채권 | 입금등록, 입금대사(부분상계), 거래처원장, 미수금·연체, 수금예정표 |
| 매입·수입 | 해외수입 LOT, 수입원가 배부(Landed Cost), 외화·환율 |
| 재고관리 | 현재고, 입출고원장, LOT·유통기한, 재고조정, 안전재고 |
| 지출·경비 | 일반경비, 고정비, 지급예정표 |
| 회계 | 자동분개, 계정별원장, 현금출납장, 손익계산서, 월마감 |
| 분석 | 매출·이익·거래처(파레토)·품목·지역·할인·재고·현금흐름 |
| 기준정보 | 거래처(별칭 통합·중복검사), 품목, 가격표, 할인정책, 계정과목, 권한 |
| 시스템 | 엑셀 가져오기, 변경이력, 백업, 와이어프레임, 버전이력 |

## 특징

- **전표 기반 데이터 구조** — 월별 시트 없음. 단일 거래 테이블 + 기간 필터
- **FIFO 원가 계산** — LOT별 개당 원가로 선입선출 소진, 매출원가 자동 확정
- **드릴다운** — 대시보드 숫자 클릭 → 원본 전표까지 역추적
- **거래처 별칭 통합** — 오타·영문·약어 표기를 하나의 코드로 병합, 유사도 경고
- **테마 4종** — 다크 / 라이트 / 블루 / 그린
- **단축키** — F2 조회 · F8 저장 · F9 신규 · Ctrl+K 검색
